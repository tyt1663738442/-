# 2026-05-16 工作日志

## 小市值股票策略前端优化

### 分时图塌陷修复
- **问题**: MinuteChart 价格图区域塌陷，数据堆积在左上角/顶部，只有成交量柱子显示
- **根因**: ResizeObserver 计算的像素高度在某些 flex 嵌套场景下不稳定，导致价格图高度异常
- **修复**: 去掉 ResizeObserver，改为与 KLineChart 一致的纯 flex 布局
  - 价格图区域: `flex-1 min-h-0`（自适应填充剩余空间）
  - 成交量区域: `shrink-0 height: 22%`（固定比例）
- **文件**: `frontend/src/components/MinuteChart.tsx`

### K线图对齐优化
- **问题**: 日线/周线 K 线蜡烛与下方成交量柱子在 X 轴上可能存在微小偏移
- **修复**: 在成交量图也添加隐藏的右侧 YAxis（`yAxisId="volRight"` + `tick={false}`），确保价格图和成交量图的内部左右空间结构完全一致，plot 区域宽度完全相同
- **文件**: `frontend/src/components/KLineChart.tsx`

### 导航栏重构 + 新增复盘分析页面
- **删除**: 从 App.tsx 导航栏中移除 行情看板、策略回测、大单追踪 三个 tab 及其相关组件引用
- **新增**: 复盘分析 (ReviewPanel) tab，基于连板质量矩阵评分规则实现
- **评分规则实现**:
  - 连板基础分: 1板5分 → 5板+18分
  - 题材评分(封顶30): 质地分(大题材10/中等8/小题材3) + 热度分(按涨停数1~8) + 龙头加分(最高板+8, 梯队≥3层+5) + 多题材叠加(2题材+2~5, 3题材+2~8)
  - 资金流向: -5~+10 (主力净流入/成交额)
  - 总分区间: 25~72分，评级 S/A/B/C
- **UI设计**: 深色科技风，顶部统计卡片 + 评分规则说明 + 可展开详情的评分表格 + 评级筛选器
- **文件**: `frontend/src/App.tsx`, `frontend/src/components/ReviewPanel.tsx`, `frontend/src/components/SectorPanel.tsx`
- **构建验证**: `npm run build` 通过

### 复盘分析对接真实数据
- **后端接口**: `backend/main.py` 新增 `/api/review/stocks` 端点
  - 自动从 `AUCTION_CACHE` 过滤连板股(boards >= 1)，缓存过期触发重扫
  - 龙头判定: 板块内最高连板数
  - 梯队深度: 板块内连板层数

- **前端对接**: ReviewPanel.tsx 改为 fetch API，每30s自动刷新，支持手动刷新按钮
- **文件**: `backend/main.py` (新增 `_calc_review_scores` + `/api/review/stocks`), `frontend/src/components/ReviewPanel.tsx`

### 复盘分析连板数修复 + 涨幅列新增
- **问题**: 连板数据全是1板，因为 `_count_boards()` 只用当天会话历史，没有真实连板数
- **修复方案**: 使用 `akshare.stock_zt_pool_em(date=date)` 获取东方财富涨停池，其中 `连板数` 字段是官方计算的
- **后端修改** (`backend/main.py`):
  - 新增 `BOARD_COUNT_CACHE` 全局变量（code → 连板数）
  - 新增 `_update_board_count_cache()` 函数，从涨停池获取并更新缓存
  - 修改 `_refresh_auction_cache()`，在构建缓存时用 `BOARD_COUNT_CACHE` 更新 `board_count`
  - `dadan_background_scan` 中每30轮（约15分钟）调用一次 `_update_board_count_cache()`
  - 首次更新通过 `auction_cache.json` 持久化文件（需要先停止后端再手动刷新文件）
- **前端修改** (`frontend/src/components/ReviewPanel.tsx`):
  - 新增"涨幅"列（`change_pct`），显示涨跌幅百分比
  - 表格改为14列布局（`repeat(14, minmax(0, 1fr))`）
  - 涨幅列支持点击排序（`SortColumn` 新增 `'change_pct'`）
  - 涨幅颜色：涨红跌绿（中国A股惯例）
- **验证**: 当前数据（2026-05-15 收盘后）: 6连板(蒙娜丽莎)、5连板(利仁科技)、3连板(京能/威龙)、2连板(北自/和远)
